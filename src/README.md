# Plugin.Maui.TvDPad

Plugin.Maui.TvDPad (formerly Plugin.Maui.DPad) provides directional pad (D-Pad) and focus/navigation helpers for .NET MAUI apps, with platform-specific integrations for Android TV, tvOS, and TV-capable platforms.

## Renaming
The project and NuGet package were renamed from `Plugin.Maui.DPad` to `Plugin.Maui.TvDPad`. Update any references in your projects, documentation, and samples accordingly.

## Quick Start
1. Add the `Plugin.Maui.TvDPad` NuGet package to your .NET MAUI project.
2. Initialize any platform-specific setup if required by your platform.
3. Subscribe to focus/navigation events and handle `DPadKey` events.
